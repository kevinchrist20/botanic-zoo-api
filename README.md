# Botanic Zoo API

Using Nodejs, Axios, and Cheerio, this app scrapes animal and plant facts from the web.

## Features
- Get animal info and facts
- Get list of pets(Dog, Cat, etc.)
- Get plant info and facts
- Developer API

## 
Botanic Zoo is free to use without cost.
